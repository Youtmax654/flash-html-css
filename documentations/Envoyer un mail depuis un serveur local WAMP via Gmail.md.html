Envoyer un mail depuis un serveur local WAMP via Gmail

Testez la fonction mail de PHP

<?php
$to = 'monAdresse@gmail.com';
$subject = 'Mail de test';
$message = "Ceci est un mail de test.";
$headers = "From: monAdresse@gmail.com";

mail($to, $subject, $message, $headers);
?>

Nous n'avons pas de service SMTP pour envoyer des mails sur notre serveur de développement WAMP, alors vous devriez obtenir l'erreur suivante :

Nous allons donc déléguer l'envoi de mail à un service SMTP externe ; celui de Gmail.

Création d'un mot de passe d'application Google

  1. Rendez-vous sur www.google.com

  2. Allez dans la gestion de votre compte Google

  3. Rendez-vous dans la section "Sécurité"

  4. Puis dans "Validation en deux étapes"

  5. Enfin dans "Mots de passe des applications"

  6. Cliquez sur "Sélectionnez une application"

  7. Choisissez "Autre"

  8. Choisissez un nom d'application

  9. Cliquez sur "Générer"

  10. Copier le code d'application et gardez le précieusement, si vous le perdez, vous devrez recommencer la procédure

  11. Cliquez sur "Ok"

  12. Rendez-vous sur mail.google.com/mail/u/0/#inbox

Configuration de Sendmail

  1. Décompressez le contenu du de l'archive sendmail.zip dans le dossier C:\wamp\sendmail\

  2. Ajoutez les configurations suivantes dans le fichier C:\wamp\sendmail\sendmail.ini

smtp_server=smtp.gmail.com
smtp_port=465
smtp_ssl=ssl

default_domain=localhost

error_logfile=error.log
debug_logfile=debug.log

auth_username={votreAdresse@gmail.com}
auth_password={votreMotDePasseDApplicationGoogle}

hostname=localhost
  • Remplacez {votreAdresse@gmail.com} par votre adresse Gmail
  • Remplacez {votreMotDePasseDApplicationGoogle} par votre mot de passe d'application Google

Délégation de l'envoi de mail à Sendmail

  1. Cliquer sur l'icone WAMP dans la barre des tâches, puis sur "PHP", puis sur "php.ini"

  2. ajouter la ligne de configuration sendmail_path = "C:\wamp\sendmail\sendmail.exe -t" à la fin du fichier

Testez à nouveau la fonction mail de PHP

<?php
$to = 'monAdresse@gmail.com';
$subject = 'Mail de test';
$message = "Ceci est un mail de test.";
$headers = "From: monAdresse@gmail.com";

mail($to, $subject, $message, $headers);
?>

Pas d'erreur ? Le mail est bien envoyé 📨
